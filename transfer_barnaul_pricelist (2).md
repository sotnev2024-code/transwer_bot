# Прайс на трансфер из Барнаула

Тариф для расчета стоимости в таблице — 30 руб./км. Расстояния по основным направлениям были собраны из открытых сервисов маршрутов, а маршрут до Телецкого озера обновлен по пользовательскому значению 465 км [cite:1][cite:9][cite:7][cite:18][cite:28][cite:31][cite:39][cite:40].

## Примечание

Стоимость рассчитана по формуле: расстояние × 30 руб./км. Для Аи сохранено ориентировочное значение 285 км из предыдущей версии прайса, которое лучше уточнять под конкретную точку назначения [cite:18].
